# 🔧 MEF Python - Método dos Elementos Finitos

Este projeto tem como objetivo implementar, do zero, um solver do Método dos Elementos Finitos (MEF) utilizando Python, com foco em problemas 1D de barras.

---

## 📌 Funcionalidades atuais

- Montagem da matriz de rigidez global
- Montagem do vetor de forças
- Estrutura modular do código (montagem, solver, pós-processamento)
- Testes básicos de validação
